Metodología de Trabajo

Para el desarrollo de este proyecto, hemos seguido una metodología de trabajo colaborativa basada en los siguientes principios:

  • División de tareas: Cada miembro del equipo se especializó en un servicio específico, permitiendo un enfoque profundo en cada área.
  • Documentación exhaustiva: Todos los procesos y configuraciones fueron documentados detalladamente para facilitar el mantenimiento futuro.
  • Pruebas cruzadas: Cada implementación fue vista por los demas miembros del equipo.
  • Integración continua: Utilizamos nuestro propio servidor Git para mantener un control de versiones eficiente de toda la documentación y código.
