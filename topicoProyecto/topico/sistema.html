Arquitectura del Sistema

Kernel Linux 6.x, init con systemd, y entornos de escritorio GNOME, KDE y XFCE. Soporte para ext4, btrfs, xfs y más.

Terminal y Comandos Básicos

Herramienta principal para administración. Comandos esenciales:

  • sudo apt update
  • sudo apt install <paquete>
  • ls, cd, mv, cp, rm

Gestión de Paquetes

APT para paquetes .deb, más Snap y Flatpak para aplicaciones en contenedores.
