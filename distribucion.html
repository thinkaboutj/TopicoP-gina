¿Qué es una Distribución Linux?

Es un conjunto de software que incluye el núcleo de Linux, herramientas de sistema, aplicaciones, y un sistema de gestión de paquetes. Cada distribución tiene su propio enfoque y conjunto de características, lo que las hace adecuadas para diferentes necesidades y usuarios.

Lanzamientos y Versiones

Ubuntu publica nuevas versiones cada 6 meses (abril y octubre). Las LTS cada 2 años con 5 años de soporte.

  • Última LTS: Ubuntu 24.04 “Focal Fossa”
  • Próxima LTS: Ubuntu 26.04 (Abril 2026)
  • Versiones intermedias: 24.10, 25.04, 25.10…

Respaldo

Es importante respaldar archivos y configuraciones esenciales del sistema. A continuación, se presenta un script que comprime y guarda la carpeta personal del usuario con fecha y hora.

Elementos a respaldar:

  • Carpeta personal del usuario (/home/usuario)
  • Configuraciones del sistema
  • Archivos de servicios importantes

Script de respaldo:


#!/bin/bash

# Carpeta a respaldar
ORIGEN="/home/usuario"

# Carpeta destino
DESTINO="/home/usuario/respaldo"

# Fecha actual
FECHA=$(date +"%Y-%m-%d_%H-%M-%S")

# Nombre del archivo
ARCHIVO="respaldo_$FECHA.tar.gz"

# Crear carpeta si no existe
mkdir -p "$DESTINO"

# Generar respaldo
tar -czvf "$DESTINO/$ARCHIVO" "$ORIGEN"

echo "Respaldo guardado en $DESTINO/$ARCHIVO"

Puedes automatizar este script con cron para realizar respaldos periódicos.
